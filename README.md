# 🎬 CineCam

> The camera mod Bedrock never had — until now.

CineCam brings **cinematic camera controls** to Minecraft Bedrock Edition. Switch between first-person cinematic view and third-person shoulder views — all with smooth interpolation and subtle camera sway.

---

## ✨ Features

- 🎥 **Cinematic First Person** — smooth camera with subtle sway and lag, feels like a real movie
- 👁️ **Right Shoulder View** — third-person camera over the right shoulder
- 👁️ **Left Shoulder View** — third-person camera over the left shoulder
- 🔄 **Hold Shift to Switch** — hold Shift and release at the right time to change mode
- 📊 **Progress Bar** — visual indicator shows which mode you'll activate
- ⚡ **Smooth Interpolation** — camera smoothly follows your movement, no jerky cuts

---

## 🕹️ How to Switch Modes

Hold **Shift** and release at the right time:

| Hold time | Mode |
|-----------|------|
| < 0.5s | Nothing |
| 0.5s – 2s | 🎥 First Person (Cinecam) |
| 2s – 3s | 👁️ Right Shoulder |
| 3s – 4s | 👁️ Left Shoulder |
| ≥ 4s | ❌ Turn OFF |

---

## 📋 Requirements

- Minecraft Bedrock Edition **1.21.90+**
- **Experimental APIs: ON**

---

## 📥 Installation

1. Download `CineCam.mcaddon`
2. Open the file — Minecraft will install it automatically
3. Create or open a world
4. Go to **Behaviour Packs** → enable **CineCam**
5. Enable **Experimental APIs** in world settings
6. Done! 🎉
